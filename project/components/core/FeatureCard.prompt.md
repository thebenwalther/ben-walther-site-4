Outcome / benefit card with a rounded gold-tinted icon tile, title, and copy. Use in a "what changes" grid.

```jsx
<FeatureCard icon="◎" title="You get clear">
  You finally know what you actually want — not what you were told you should.
</FeatureCard>
```

`tone="dark"` for forest sections. The icon can be a unicode glyph or any node.
