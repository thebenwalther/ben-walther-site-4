Pill-shaped brand button — use the gold `accent` variant for the single primary CTA on a view, and `outline`/`soft`/`dark` for secondary actions.

```jsx
<Button variant="accent" iconRight="→">Book a free intro call</Button>
<Button variant="outline">See how it works</Button>
```

Variants: `accent` (gold, primary), `dark` (forest), `light` (white), `outline`, `outline-on-dark` (for dark heroes), `soft` (tinted gold). Sizes: `sm` `md` `lg`. Props: `href` (renders an `<a>`), `iconLeft`/`iconRight`, `fullWidth`, `disabled`. Hover lifts the button 2px.
