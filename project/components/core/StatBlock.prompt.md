Big confident proof numeral with a caption — the trailing unit renders in gold. Use in stat strips on hero and social-proof bands.

```jsx
<StatBlock value="400" unit="+" label="clients coached" />
<StatBlock value="4.9" unit="★" label="average rating" tone="dark" size="lg" align="center" />
```

`tone="dark"` for forest backgrounds. Sizes `sm` `md` `lg`. `align` left or center.
